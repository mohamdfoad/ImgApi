Udacity Image Resize API Project

Listening at /api/img for queries containing at least a valid image filename , width and height

Examples:


/api/img?filename=santamonica&width=320&height=210

/api/img?filename=encenadaport&width=320&height=210

/api/img?filename=palmtunnel&width=320&height=210

/api/img?filename=icelandwaterfall&width=320&height=210

/api/img?filename=fjord&width=320&height=210
